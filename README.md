# Calculadora de Alcance de Anúncios

## Cenário
A agência **Divulga Tudo** precisa de um programa para gerenciar os seus anúncios online. O objetivo dos anúncios faz parte de uma campanha nas redes sociais. O sistema de gerenciamento permitirá a gestão do anúncio e o rastreio dos resultados da campanha.
## Requisitos
O sistema fornecerá os relatórios de cada anúncio contendo:

 - valor total investido
 - quantidade máxima de visualizações
   
   
 - quantidade máxima de cliques
       
 - quantidade máxima de compartilhamentos

          

 - Os relatórios poderão ser filtrados por intervalo de tempo e cliente.

## Execução do Projeto

Baixe o arquivo ZIP https://github.com/dev-MarcioCosta/DesafioCapgemini_CalculadoraDeAlcance/archive/refs/heads/master.zip, extraia e abra na IDE de sua preferência.

Para esse projeto utilizei a IDE Eclipse.

Passo-a-Passo no Eclipse:

Com o arquivo extraído no seu local de preferência, na IDE Eclipse clique em **file** --> **open projects** --> **directory**  -->**selecione a pasta** --> **finish**.
Agora, expanda a pasta e clique sobre **DesafioCapgemini_CalculadoraDeAlcance-master/src** para abrir, agora expanda o pacote application e abra a classe Sistema e  à execute.




## Utilizando o Sistema

Após  iniciar a execução do programa, o usuário deve interagir com o sistema digitando o número da função correspondente que deseja e preencher as informaçôes corretamente conforme são indicadas.
